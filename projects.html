My own keyboards

Searching for the perfect keyboard is not only a time-consuming process but also expensive. A project since 2019, I currently have made and own 4 different keyboards, with different switches and keycap profiles.

Favourite switches:

  1. C3 Banana Split (Linear)
  2. Okomochi (Tactile)
  3. Silent Alpaca (Linear)

Macro Keypads

Are making keyboards really complete without attempting to make my own macro keypads? I've made 3 macropads with just a 3D printer and Arduino micros.

Using the Arduino IDE to map each key on the macropad to a corresponding key on the computer (such as F13-F24), and using AutoHotKeys to make macros, I've been able to make some gaming experiences easier and help with sorting out my windows on the desktop.
